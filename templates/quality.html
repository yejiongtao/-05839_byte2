Quality: About This Data Set

The dataset is not perfectly complete. It has some missing data. But it's coherent, correct and accountable.

Is the data complete?

The dataset has some missing data. For example, the 'Deaths' column has some missing data (as shown below), which is shown when I used Wrangler for data manipulation.

The missing data definitely has influence in my analysis, because my goal is related to the number of death. To figure out what those missing data are, I filtered the 'Leading Cause' column. As shown below, the death number of the same cause should be similar. Thus, I believe those missing data should be 0's.

As for other columns, some numerical data is missing as well, but not related to my analysis.

Is the data coherent?

Yes, the data makes sense. For example, the distribution of 'Leading Cause' column is as follows. Ranging from 0.09% to 8.76%, which is normal.

And the Sex and Race Ethnicity columns also make sense. As shown below, people from different groups are of similar numbers.

Is the data correct?

To some extent, the data should be correct, because it's provided by Department of Health and Mental Hygiene (DOHMH). If the department's datasource is reliable, then this dataset should be correct. While there could be many causes that might endanger the correctness of it. For example, how the data collected? Is it reported by all the hospitals in New York City? Is it possible for hospitals to make mistake? However, I would assume its correctness, because of its source should be a relatively reliable one.

Is the data accountable?

Yes. I know the source of the data, which is reliable. I have stored the original version and changed versions of data. Only myself can change the data displayed in this website. So it's accountable.
